🔑 GitHub Token einrichten

Einmalige Einrichtung — danach veröffentlichen Sie mit einem Klick.

Schritt 1 — Token erstellen

Ein Personal Access Token erlaubt dem Admin-Portal, Dateien direkt in Ihr GitHub Repository zu schreiben — ohne dass Sie etwas herunterladen oder hochladen müssen.

So erstellen Sie den Token:
  1. Auf github.com/settings/tokens/new gehen
  2. Note: ibox-studio-admin
  3. Expiration: No expiration wählen
  4. Unter repo → alles anhaken
  5. Generate token klicken
  6. Token kopieren (beginnt mit ghp_)
🔗 GitHub Token erstellen ↗

Schritt 2 — Token eingeben

Der Token wird sicher in Ihrem Browser gespeichert und verlässt Ihren Computer nie unverschlüsselt.
